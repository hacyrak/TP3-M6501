Ce projet est le TP3-M6501 du programme LP-CODESI 2018/2019
Il est basé sur l'utilisation de GIT.
Pour cela, on a créer un dossier TP3-M6501
On a initialisé GIT :(git init)
Ajouter un fichier
Commit un fichier (git add fichier + git commit -m "Premier commit")
Modifier le fichier
Commit (git add fichier + git commit -m "Commit modifier")
Ainsi git log nous donne 2 commit
